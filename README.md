## About This Project
Performed deep dive analysis on Heart Disease 
Dataset and built an interactive dashboard 
using Looker Studio.

## Dataset
- Total Patients: 303
- Total Features: 14
- Target: Heart Disease (Yes/No)

## KPIs Defined
- Heart Disease Rate
- Average Age of Patients
- Average Cholesterol Level
- Average Max Heart Rate
- Gender Distribution

## What I Did
- Loaded and cleaned Heart Disease Dataset
- Defined 5 Core KPIs
- Performed Patient Segmentation using KMeans
- Created Risk Level Categories (Low/Medium/High)
- Built Interactive Dashboard in Looker Studio
- Identified Key Business Insights

## Key Insights
- Heart disease found in older patients
- High cholesterol linked to high risk
- 3 clear patient segments identified
- Average age of patients is 54 years
